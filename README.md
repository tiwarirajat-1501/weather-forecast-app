# 🌦 Weather Forecast App

A modern and responsive Weather Forecast Web Application built using **Flask**, **HTML**, **CSS**, and **JavaScript**.  
This app provides real-time weather updates along with a 5-day forecast using the OpenWeatherMap API.

---

# 🚀 Features

✅ Real-time Weather Data  
✅ 5-Day Weather Forecast  
✅ Dynamic Weather-Based Backgrounds  
✅ Responsive Modern UI  
✅ Animated Weather Icons  
✅ Loading Spinner Animation  
✅ Sunrise & Sunset Timings  
✅ Feels Like Temperature  
✅ Humidity, Pressure & Wind Speed  
✅ Mobile Friendly Design  
✅ Smooth Hover & Fade Animations

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/weather-forecast-app.git
```

---

## 2️⃣ Open Project Folder

```bash
cd weather-forecast-app
```

---

## 3️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Add Your API Key

Get your API key from:

https://openweathermap.org/api

Replace this line in `app.py`:

```python
API_KEY = "YOUR_API_KEY"
```

with:

```python
API_KEY = "your_actual_api_key"
```

---

## 5️⃣ Run Flask App

```bash
python app.py
```

---

## 6️⃣ Open In Browser

```plaintext
http://127.0.0.1:5000
```
